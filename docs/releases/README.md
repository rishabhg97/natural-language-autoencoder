# Release Documents

This directory separates public-release requirements from experiment status.
A checkpoint can be scientifically useful while redistribution, replication,
or a broader claim remains blocked.

- `r33_clean_sft_publication_checklist.md`: required evidence and release gates.
- `r33_clean_sft_model_card_draft.md`: claim-scoped model card for the selected
  R33 supervised AV+AR pair.
- `r33_clean_sft_license_provenance.md`: base-model, source-data,
  teacher-output, and redistribution terms inventory.
- `r33_clean_sft_NOTICE.txt`: incomplete attribution template for owner/legal
  review; not a final notice.
- `r33_clean_sft_human_review.md`: blinded semantic-review handoff and scoring
  protocol.
- `r33_clean_sft_public_bundle_readme.md`: claim and blocker notice embedded in
  the redacted, no-weights release candidate.
- `r33_clean_sft_release_candidate_attestation.md`: exact archive, security,
  S3-preservation, scientific-result, and remaining-gate index maintained
  outside the immutable candidate archive.
